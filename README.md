# 📦 Data Structures in C (From Scratch)

## 📖 Description
Ce projet a pour objectif d’implémenter **les structures de données fondamentales en langage C from scratch**, sans utiliser de bibliothèques externes.

L’objectif est de :
- Comprendre en profondeur le fonctionnement interne
- Maîtriser la gestion de mémoire
- Améliorer les compétences en algorithmique et en C

---

## 🚀 Structures implémentées

- 🔗 Singly Linked List
- 🔗 Doubly Linked List
- 📚 Stack (Pile)
- 📥 Queue (File)
- 🔄 Deque
- ⭐ Priority Queue
- 🌲 Heap (Min / Max)
- 🌳 Binary Search Tree (BST)
